# MindFool.Games - Specification Documentation

**A mindful mobile game for focus and calm through traditional attention practices.**

---

## Overview

This repository contains the complete specification documentation for MindFool.Games, following the [agent-os methodology](https://github.com/buildermethods/agent-os) for AI-assisted development.

**Platform:** Android-first (Expo React Native → PWA via RN Web)
**Session length:** 2–10 minutes
**Audience:** Anyone seeking calm alertness, mindful fun, and focus through traditional-inspired micro-games

---
